# Template for React Typescript Vite.js NetCore7

### Prerequisites
- npm
- net7 runtime

### **1. Create app template**
- npx @virtual-registry/rtvnet7 YourAppName

### 2. **Install clientapp dependencies**
- cd YourAppName  
- cd clientapp  
- yarn or npm i

### 3. **Start your app**
- Start AspNet app  
  `you can use visual studio` or `dotnet run start` command
- Start clientapp  
  `cd clientapp`  
  `yarn run dev`
